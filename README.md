## 9节点 LangGraph 流水线 / 9-Node LangGraph Pipeline

```
identify → disassemble → decompile → cfg → crypto → c2 → evasion → strings_imports → report
```

| 节点 | 功能 |
|---|---|
| identify | 识别二进制格式/平台/架构 |
| disassemble | 反汇编生成汇编代码 |
| decompile | 反编译为伪C代码 |
| cfg | 生成控制流图 |
| crypto | 检测密码学例程 |
| c2 | 分析 C2 通信协议 |
| evasion | 检测规避技术 (MITRE ATT&CK) |
| strings_imports | 提取字符串与导入表 |
| report | 生成综合逆向分析报告 |
